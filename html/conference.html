03. UX 전략 및 설계 원칙

  • 세션 인사이트 기록 → 공유 → 교류까지 이어지는 핵심 흐름 중심 UX
  • MVP 우선 설계 및 확장 가능성 고려

디자인 시스템

  • Foundation: 색상, 그리드, 타이포 시스템 정의
  • Components: Slot, Tab, Card 등 반응형 대응 컴포넌트 설계
  • UI 일관성: 세션, 노트, 마이페이지 동일한 구조 및 인터랙션
  • 재사용성 강화: 공통 UI 컴포넌트화

04. UX 플로우 및 정보구조 설계

핵심 플로우

세션 탐색
세션 신청
인사이트 노트 작성
마이페이지 공유

정보구조 (IA)

  • 홈: 세션 안내 + 인사이트 노트 강조
  • 세션: 키워드 필터 + 카드형 요약
  • 노트: 실시간 저장 + 공유 기능
  • 마이페이지: 신청내역 및 작성 콘텐츠 확인
